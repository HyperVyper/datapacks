## sapphire

Sapphire gems will be generated in the ocean deep down. Detect weather, attract lightning and create decorative particles.

*S - Sapphire*
*G - Glass*
*I - Iron*
*B - Bucket*

**Weather Detector:**
```
GGG
ISI
IBI
```

**lightning Rod:**

