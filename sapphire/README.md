## sapphire

Sapphire gems will be generated in the colder oceans, deep down. Detect weather, attract lightning and create decorative particles.
Recommended to use with limitedlife.

*S - Sapphire*

*G - Glass*

*I - Iron*

*B - Bucket*

**Weather Detector:**
```
GGG
ISI
IBI
```

**lightning Rod:**
```
 S
ISI
ISI
```
